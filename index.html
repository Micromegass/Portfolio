EDUCATION

University of Hagen (ongoing)

Realizing that I love to create things, solve problems and always learn something new, I decided to enroll again in the distance university of Hagen to major in Computer Sciences. The university is the only state universtiy in Germany that allows you to study in different countries and take your exams in the German schools around the world. It also allows you to study at your own rhythm while working fulltime. It is a normal Bachelor of Science programme in which you learn all there is to know about the fundamentals of Computer Sciences

Make it Real Camp (graduated)

I took part in an intensive instructor-led coding Bootcamp at Make it Real. The programme included three phases. The first phase included self-study and an entrance test. Once accepted, the second phase started which consisted of intensive 12-week full-time programme. Everyday students and the teacher came together to study Ruby, Rails, Sinatra, Javascript, jQuery, APIs, Testing, Bootstrap, HTML, CSS and various other such as responsive design, testing techniques and deploying different services such as Heroku and Amazon Web services. During post-phase students were assigned a mentor and asked to complete the rest of the online platform which is full of material, coding challenges and projects.

University of Friedrich-Alexander (graduated)

Because I was interested in the workings of the economy I studied Economics and Anglistic at the university of Friedrich-Alexander. Apart from Economics my degree involved a broad variety of subjects. I studied critical theory and I also enrolled in spanish and philosophy classes.

Picas y Fijas

Tools: Javascript, jQuery

Picas y Fijas

What's it all about?

This project was my first project using pure Javascript and jQuery for the backend. The website generates a random 4-digit number which the user has to guess. Each time the user enters a 4-digit number the website tells the users how many picas (correct numbers, but wrong place) and how many fijas (correct number at the right place) the input number has. The user keeps on guessing until the right number is found. Once guessed correctly the user can pick up a new game.


Likes and challenges creating the project:

I really enjoyed it because it was a tricky project. I learned how to use Javascript and jQuery better.

Ruby on Crack

Tools: Ruby, RoR

Ruby on Crack

What's it all about?

This application is a forum where ruby programmers can ask questions about Ruby and Ruby on Rails. Much like Stackoverflow, yet the focus is only Ruby and its frameworks. Once signed up and authenticated, users can ask new questions and answer questions. Plus, they can comment and vote for for answers and questions.


Likes and challenges creating the project:

This project took me some time and it was a real challenge. I had to use everything I knew about Ruby and do a lot of research. The most difficult part was to create a polymorphic voting system and a polymorphic comment system. I also tried to not use the gem Devise to create user authentication which also had its challenges. But at the end I am very happy with the application and I think the styling with CSS makes the application and the UI just very nice.

Cities

Tools: Ruby, RoR, Heroku

Cities

What's it all about?

The idea of this application is that everyone who is registered and authenticated can create new posts about cities or places. When a new post is created the user can upload an image which is the background of his post. Other users can comment beneath the post.


Likes and challenges creating the project:

The Backend part for this project was easier than the deployment part. I tried to use AWS and Heroku for deployment what only worked to some degree. After a certain amount of time, Heroku unfortunately deletes the uploaded pictures. A problem, which I could not yet solve. That's why I added a fallback background which is a workable solution but not the optimum. Another challenge was to configure the whole app in Spanish with the gem of Sven Fuchs.
