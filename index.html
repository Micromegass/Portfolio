Picas y Fijas

Tools: Javascript, jQuery

Picas y Fijas

What's it all about?

This project was my first project using pure Javascript and jQuery for the backend. The website generates a random 4-digit number which the user has to guess. Each time the user enters a 4-digit number the website tells the users how many picas (correct numbers, but wrong place) and how many fijas (correct number at the right place) the input number has. The user keeps on guessing until the right number is found. Once guessed correctly the user can pick up a new game.


Likes and challenges creating the project:

I really enjoyed it because it was a tricky project. I learned how to use Javascript and jQuery better.
